# Plumber Quest 1985

A browser-based retro platformer inspired by classic 1980s side-scrollers.

## Notes

- This project intentionally uses original code and placeholder visuals.
- It is inspired by old-school platform game mechanics, but is not an exact clone.

## Run

Open `index.html` directly in a browser, or use a local static server.

### Optional local server (Node.js)

```bash
npx serve .
```

Then open the URL shown in the terminal.

## Controls

- Left: Arrow Left or A
- Right: Arrow Right or D
- Jump: Arrow Up or W or Space
- Mobile: On-screen buttons

## Goal

Collect coins, avoid enemies, and reach the flag before time runs out.
