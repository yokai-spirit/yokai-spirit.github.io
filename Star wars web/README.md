# Star Wars Web App

A simple Node.js web application that fetches data from the [Star Wars Databank API](https://starwars-databank.vercel.app/) and displays it as cards grouped by categories.

## Features
- Node.js + Express backend
- Fetches data from an external API
- Displays cards with images and information
- Groups cards by category
- User interface in English

## Getting Started

1. Install dependencies:
   ```sh
   npm install
   ```
2. Start the server:
   ```sh
   npm start
   ```
3. Open your browser and go to [http://localhost:3000](http://localhost:3000)

## Project Structure
- `server.js` — Express server
- `public/` — Static frontend (HTML, CSS, JS)
